Delicate weather app

Delicate is a real-time weather web app built from scratch through my coding knowledge as listed: HTML, CSS, JavaScript, API. What makes the app even more special that I designed and prototyped the interface beforehand. The interface of the app was made using Sketch and Figma, as well as all the illustration and the icons all specially crafted for this app. The unique selling point of the app is that the interface changes as the weather condition changes such as, Sunny, Partly Cloudy, Cloudy, Raining, and Snowing. You can find all five of them by looking for various cities.

Tools: Sketch, Figma, Visual Studio Code, Openweather API.
